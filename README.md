# RNA_sequencing_code
## Input

Reads files: fastq

Genome: fasta

Annotation: gtf

## Pipelines

### 0 QC fastqc

### 1 STAR index & mapping

### 2 featurecounts quantification

### 3 DEG_pre-process -R

### 4 DEG_DESeq2 -R

### 5 Visualization -R (heatmap, PCA, correlation, volcano plot)
